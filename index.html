Fishing Selection Guide

Overview

This guide provides recommendations for selecting appropriate fishing tools based on fish type, well conditions, and operational constraints.

Key Selection Criteria:

  • Fish type and dimensions
  • Wellbore geometry and condition
  • Fishing environment (cased vs. open hole)
  • Available fishing tool inventory
  • Operational constraints and risks

Fishing BHA Design

Standard Fishing BHA Components

Top Section (Surface to Jar)

  • Drill Pipe or Workstring
  • Accelerator (if required)
  • Drilling Jar

Middle Section (Jar to Fishing Tool)

  • Drill Collars or Heavy Weight Drill Pipe
  • Stabilizers (if required)
  • Knuckle Joint (for deviated wells)

Bottom Section (Fishing Tools)

  • Fishing Tool (Overshot, Spear, etc.)
  • Grapple/Die Collar
  • Mill or Cutting Tool (if required)

BHA Design Considerations

Wellbore Factors

  • Hole size and casing program
  • Wellbore trajectory and dogleg severity
  • Formation type and stability
  • Mud properties and hydraulics

Operational Factors

  • Expected fishing loads and jarring forces
  • Available rig equipment capabilities
  • Fishing tool specifications and limitations
  • Risk assessment and contingency planning

Design Best Practices

  • Include adequate weight for effective jarring
  • Use appropriate stabilizers for wellbore control
  • Consider fishing tool access and engagement requirements
  • Plan for contingencies with backup tools
  • Verify tool compatibility and connection types

Common BHA Configurations

Standard Configuration: DP + Jar + DC + Fishing Tool

Deviated Well: DP + Jar + KJ + DC + Fishing Tool

High-Risk Fishing: DP + Accelerator + Jar + DC + Fishing Tool

Fish-Specific Considerations

Packer Fishing

  • Verify packer setting mechanism before fishing
  • Consider chemical cutters for difficult retrievals
  • Use spear for internal fishing of packer bore
  • Milling may be required if conventional methods fail

Tubular Fishing

  • Determine if fishing internal or external is preferable
  • Consider washover if fish is stuck or cemented
  • Use overshot for external fishing of pipe
  • Use spear for internal fishing of pipe

BHA Fishing

  • Strong jarring assembly typically required
  • Consider accelerator for additional impact
  • Overshot is primary tool for drill collars
  • May require milling if fish is severely damaged

Junk Fishing

  • Use junk baskets for small, loose pieces
  • Magnets effective for ferrous materials
  • May require multiple fishing runs
  • Consider milling if junk cannot be retrieved

Best Practices & Safety Guidelines

Safety Considerations

  • Always conduct thorough risk assessment before fishing operations
  • Ensure proper well control equipment is functional
  • Establish clear communication protocols
  • Monitor well pressures continuously
  • Have emergency response plans in place

Operational Efficiency

  • Document all fishing operations thoroughly
  • Monitor torque and drag trends carefully
  • Use appropriate jarring procedures
  • Establish clear success criteria
  • Know when to change fishing strategy

Troubleshooting Common Issues

Fish Cannot Be Engaged
  • Verify fish depth and condition with caliper log
  • Consider different fishing tool type
  • Check for wellbore obstructions
Fish Slips During Retrieval
  • Check grapple/die condition and size
  • Apply appropriate weight while pulling
  • Consider using different engagement method
Excessive Drag or Sticking
  • Check for wellbore geometry issues
  • Consider different BHA configuration
  • Evaluate mud properties and hydraulics
Multiple Fishing Attempts Fail
  • Re-evaluate fishing strategy
  • Consider sidetrack or abandonment
  • Consult with fishing experts
